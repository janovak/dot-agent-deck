- **Multi-line Pastes Stay Intact for Copilot CLI Agents**
  Pasting a multi-line snippet into a Copilot CLI agent pane now arrives as a single bracketed paste, instead of being submitted line-by-line as one Enter per newline. The paste handler treats Copilot CLI as if it had advertised bracketed-paste mode (DECSET 2004), even though Copilot CLI does not currently send the enable sequence itself. Claude Code and OpenCode panes — which already advertise bracketed paste — are unaffected, and plain shell panes (no agent detected) remain raw to avoid corrupting input with literal `␛[200~` text.

- **Windows-Explorer-Style Right-Click Copy and Paste in Panes**
  Highlighting text inside a focused pane now keeps the highlight visible after you release the mouse — nothing is copied yet. Right-clicking afterwards copies the highlight to the system clipboard (via OSC 52) and clears the selection. Right-clicking with no highlight pastes the system clipboard contents into the focused pane through the same path as `Ctrl+V` (trailing CR/LF stripped so the paste does not auto-submit; bracketed-paste wrap when the focused pane is a known agent CLI). Pressing `Ctrl+d` to leave PaneInput also dismisses any persistent highlight. Previously, releasing a drag-selection auto-copied immediately and the highlight vanished, which made it hard to confirm the right region was selected and added one-keystroke noise to status. The new model mirrors how cmd.exe / PowerShell / Windows Terminal behave outside dot-agent-deck.

- **Pastes No Longer Auto-Submit on Trailing Newline**
  Pasting text into an agent pane no longer auto-submits when the clipboard content ends with a newline. The paste handler now strips trailing CR/LF before forwarding to the pane, so the user must press Enter explicitly to submit. Internal newlines in multi-line pastes are preserved — bracketed-paste mode (advertised by Claude Code / OpenCode, forced for Copilot CLI) keeps them as content. Previously, copying a snippet that ended with a blank line — common when copying from a text editor or web page — would submit the paste immediately, often before the user could see what landed in the prompt.

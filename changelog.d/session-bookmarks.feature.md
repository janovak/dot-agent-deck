- **Bookmarked Copilot CLI Sessions**
  A new third tier of session management alongside `--continue` and named workspaces: a curated, manually-managed list of Copilot CLI session GUIDs with notes, designed to outlive any single workspace. Press **Ctrl+B** on a dashboard card to bookmark its session (with a one-line note prefilled from any existing bookmark); press **Ctrl+Shift+B** to open a picker that lists every bookmark and lets you resume one in a new pane via `copilot --resume <guid>`. Bookmarked cards display a **★** next to their name. CLI surface: `dot-agent-deck bookmarks list` and `dot-agent-deck bookmarks delete <prefix-or-note>`. Bookmarks live at `~/.config/dot-agent-deck/bookmarked-sessions.json`; the session name is looked up from `~/.copilot/session-store.db` at bookmark-create time (read-only) and cached, so it stays meaningful even if the underlying row is later removed.

- **Force Bracketed Paste from Launch Command When Agent Hooks Are Absent**
  When a pane is launched with a known agent CLI command (`copilot`, `claude`, `opencode`, including `.exe`/`.cmd`/`.bat`/`.ps1` variants and quoted Windows paths), dot-agent-deck now wraps pastes in bracketed-paste markers immediately — without waiting for the agent to fire its first hook event. Previously, the override depended on `SessionState::agent_type` being set, which only happens after the first hook arrives. Pastes that happened during the early window (right after spawn, or when hooks were never installed) fell through to the raw branch and got submitted line-by-line.
